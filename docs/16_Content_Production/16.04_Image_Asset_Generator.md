# Document 16.04 — Image Asset Generator

Version: 1.0
Status: Production Specification

## Purpose

Standardize image planning and asset generation for every Learn Center article.

## Objectives

- One featured image per article
- Social sharing image
- Optional diagrams and illustrations
- Consistent branding
- Accessibility-first

## Required Assets

- featured-image.webp
- og-image.webp
- twitter-image.webp
- image-manifest.json

Optional:

- diagram-01.svg
- diagram-02.svg
- comparison-table.svg
- process-flow.svg

## Image Standards

Featured Image
- 1600 × 900 px
- WebP preferred
- Clear title-safe area
- Consistent InstantViral branding

Open Graph
- 1200 × 630 px

Twitter/X
- 1200 × 675 px

## Accessibility

Every image requires:
- Alt text
- Caption (when useful)
- Descriptive filename

## Prompt Templates

Generate prompts for:
- Featured image
- Social image
- Infographics
- Comparison graphics
- Timelines

## Required Files

- image-brief.md
- image-prompts.md
- image-manifest.json

## Required Functions

- generateImageBrief()
- generateImagePrompts()
- validateImageAssets()
- buildImageManifest()

## Validation

Check:
- Required assets exist
- Dimensions match specification
- Alt text present
- No duplicate filenames
- Manifest valid

## Acceptance Checklist

- Image brief generated
- Prompt templates generated
- Manifest generated
- Accessibility requirements satisfied
- Ready for design or AI image generation
