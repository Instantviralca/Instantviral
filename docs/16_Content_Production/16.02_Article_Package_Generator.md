# Document 16.02 — Article Package Generator

Version: 1.0
Status: Production Specification

## Purpose

Automatically generate a complete production-ready package for every new Learn Center article.

## Objectives

- One command creates the entire article folder.
- Enforce the Content Design System.
- Produce identical structure for every article.
- Prevent missing files and inconsistent naming.

## Command

create-article <platform> <slug>

Example:

create-article instagram how-to-get-more-instagram-followers

## Generated Structure

01_Research.md
02_Article_Brief.md

## Template Requirements

README
- Article status
- Owner
- Last updated

Research
- Search intent
- Primary keyword
- Secondary keywords
- Competitor notes
- Search questions

Brief
- H1
- Slug
- Meta title
- Meta description
- Outline
- CTA plan

Article
- MDX frontmatter
- Placeholder headings following the Content Design System

SEO
- Meta title
- Meta description
- Canonical
- Open Graph
- Twitter
- Robots

FAQ
- JSON structure
- Schema compatible

JSON-LD
- Article schema template

Checklist
- Editorial QA checklist

## Validation

Verify:

- Valid slug
- Supported platform
- No duplicate folder
- All template files created
- Valid JSON
- Required frontmatter exists

## Required Functions

- createArticlePackage()
- createTemplateFiles()
- validateArticlePackage()
- updateContentIndex()
