# Document 16.03 — SEO Asset Generator

Version: 1.0
Status: Production Specification

## Purpose

Automatically generate SEO assets for every Learn Center article from a single source of truth.

## Outputs

- seo.json
- schema.json
- faq.json
- social metadata
- sitemap metadata

## Generate

- Meta title
- Meta description
- Canonical URL
- Robots
- Open Graph
- Twitter/X metadata
- Primary keyword
- Secondary keywords
- Reading time
- Category
- Tags

## Rules

- Unique title and description
- Self-referencing canonical
- No keyword stuffing
- Validate JSON
- Respect published/draft status
- Drafts must be noindex

## Required Functions

- generateSeoAssets()
- validateSeoAssets()
- generateOpenGraph()
- generateTwitterCard()
- generateArticleSchema()
- generateFaqSchema()
- estimateReadingTime()

## Validation

Check:

- Duplicate titles
- Duplicate descriptions
- Missing canonical
- Invalid schema
- Missing OG image
- Invalid robots
- Invalid JSON

## Acceptance Checklist

- SEO assets generated
- JSON validated
- Schema generated
- Draft protection applied
- Ready for publication
