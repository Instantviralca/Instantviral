# Document 16.05 — Social Media Asset Generator

Version: 1.0
Status: Production Specification

## Purpose

Generate reusable social media assets from each published Learn Center article.

## Supported Platforms

- Facebook
- Instagram
- X
- LinkedIn

## Outputs

- social-posts.md
- social-assets.json

## Generate

- Short teaser
- Standard post
- Long-form post
- CTA variations
- Suggested hashtags
- UTM-ready destination URL

## Rules

- Match article facts
- No clickbait
- No unsupported claims
- Consistent brand voice
- Respect platform character limits

## Required Functions

- generateSocialPosts()
- generateHashtags()
- buildCampaignAssets()
- validateSocialAssets()

## Validation

Check:

- Missing CTA
- Invalid destination URL
- Excessive hashtags
- Duplicate copy
- Character limit compliance

## Acceptance Checklist

- Platform variants created
- JSON validated
- Brand voice consistent
- Ready for scheduling
