# InstantViral v2

## Project Overview

### Vision

InstantViral is a premium social media growth platform designed to help creators, influencers, businesses and brands grow their presence across the world's largest social media platforms.

The platform currently focuses on four major networks:

- Instagram
- TikTok
- YouTube
- Facebook

The objective is to create a fast, modern, trustworthy website that feels more like a SaaS product than a traditional SMM panel.

The experience should inspire confidence from the first second a visitor lands on the website.

The website should prioritize trust, simplicity, speed, usability and excellent search engine optimization.

The design language should be inspired by premium software companies such as Stripe, Vercel, Linear, Notion and Framer.

This project should become the foundation of the InstantViral brand for the next 5–10 years.
