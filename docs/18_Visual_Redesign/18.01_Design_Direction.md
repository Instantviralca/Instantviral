# InstantViral Visual Redesign — Design Direction

## Audit summary (before)

- Rebuilt site used a restrained blue-gray brand token unrelated to the live logo.
- Large flat white sections felt empty and template-like.
- Breadcrumbs competed with H1 as a full-width block.
- Heroes relied on a single inset image without layered trust/visual composition.
- Pricing/cart/checkout were functional but visually flat (weak hierarchy, weak CTAs).

## Direction (after)

Premium Canadian social-growth brand: clean surfaces + InstantViral orange energy.
Inspiration from polished SMM checkout UX patterns only (hierarchy, spacing, sticky summary) — not brand assets.
