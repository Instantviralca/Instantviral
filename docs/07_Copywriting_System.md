# InstantViral v2

# Document 07 — Copywriting System

## Purpose
Define how every word on InstantViral should be written before creating page copy.

## Brand Voice
- Professional
- Trustworthy
- Helpful
- Clear
- Confident
- Never exaggerated

## Tone
- Simple English
- Active voice
- Short paragraphs
- Practical advice
- Honest claims

## Audience
Creators, influencers, brands, agencies and businesses seeking social media growth.

## Writing Rules
- Put the answer first.
- One idea per paragraph.
- Avoid fluff.
- Avoid keyword stuffing.
- Use descriptive H2/H3 headings.
- Write for humans first.

## CTA Rules
Primary CTA:
Get Started

Secondary CTAs:
Learn More
Explore Services
Contact Us

## Homepage Guidelines
Hero: 80–100 words
Section intros: 40–70 words
Feature cards: 20–35 words
FAQ answers: 60–120 words

## Service Page Guidelines
Hero: 100–140 words
Benefits: 60–90 words each
Feature blocks: 40–80 words
FAQ: 70–150 words

## Learn Article Guidelines
- Helpful first
- Cite reputable sources when needed
- Use examples
- Clear conclusion
- Link relevant services naturally

## SEO Writing
- One primary keyword per page
- Natural secondary keywords
- Descriptive anchor text
- Match search intent

## AI Overview Guidelines
- Answer questions directly
- Use concise summaries
- Include definitions where useful
- Keep claims factual
- Demonstrate expertise

## E-E-A-T
- Avoid unverifiable claims
- Be transparent
- Explain processes
- Build trust through clarity

## Formatting
- H1 once
- Logical H2/H3 hierarchy
- Bullets where useful
- Short paragraphs

## Acceptance Checklist
- Voice defined
- Tone defined
- CTA rules defined
- SEO writing rules defined
- AI overview rules defined
- Ready for homepage copywriting
