# Document 15.01 — Learn Center Architecture

Version: 1.0
Status: Production Specification

## Purpose

Create a scalable Learn Center that supports SEO, topical authority, and internal linking without duplicating content.

Public routes:

- /learn
- /learn/[category]
- /learn/[slug]

## Core Objectives

- Central article repository
- Category pages
- Tags
- Search
- Featured articles
- Related articles
- Related services
- Breadcrumbs
- Reading time
- Author support
- Future multilingual support

## Initial Categories

- Instagram
- TikTok
- Facebook
- YouTube
- Social Media Marketing
- Guides
- News (future)

## Data Model

Each article supports:

- id
- slug
- title
- excerpt
- content
- category
- tags
- authorId
- featuredImage
- readingTime
- publishedAt
- updatedAt
- seo
- relatedServices
- relatedArticles
- featured
- published
