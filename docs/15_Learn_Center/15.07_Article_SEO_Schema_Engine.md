# Document 15.07 — Article SEO & Schema Engine

Version: 1.0
Status: Production Specification

## Purpose

Centralized Article SEO metadata and JSON-LD schema generation for Learn articles.

Authority: this document (prompt-derived). Integrates with Documents 14.04, 14.05, 14.07, 14.08, and 15.01–15.05.

## Core Rules

- Generate metadata and schema from the shared article registry
- Never invent authors, credentials, dates, images, statistics, ratings, or sources
- Draft, preview, future-scheduled, and inactive articles must not be indexable
- Strict TypeScript; no `any`

## Canonical

`https://instantviral.ca/learn/{slug}`

## Robots

- Published + active + indexable: `index, follow`
- Draft / preview / future / incomplete / noindex: `noindex, nofollow` and excluded from sitemap

## Schema

Default: `Article`. Future: `BlogPosting`, `NewsArticle`, `HowTo` only when accurate.

FAQ schema only when FAQs are visible, approved, and `schemaEligible`.

No Review / AggregateRating on normal Learn articles.

## Acceptance

- Typecheck, lint, tests, and production build pass
