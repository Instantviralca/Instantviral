# InstantViral v2

# Document 14.01 — Global Trust Center Component

Version: 1.0
Status: Production Specification

## Purpose

Create one reusable Trust Center component that can be used on:

- Homepage
- All Service Pages
- Checkout
- About Us
- Contact
- Order Tracking

Do not duplicate trust content across pages.

---

## Component Sections

1. Trust Header
2. Trust Badges
3. Why Choose InstantViral
4. Secure Checkout
5. No Password Required
6. Refund & Refill Information
7. Customer Support
8. Final CTA

---

## Trust Badges

Render configurable badges only:

- Trusted Since 2018
- Secure Checkout
- No Password Required
- 24/7 Support
- Real Package Data
- Refill Protection (Eligible Packages Only)
- 30-Day Money-Back Guarantee (Eligible Purchases Only)

Badges must be configurable and individually enabled/disabled.

---

## Rules

- Never claim guarantees that are unsupported.
- Refill wording must come from package data.
- Money-back wording must reference the Refund Policy.
- Do not hardcode badge text inside components.

---

## Content Source

Create a shared Trust Content module:

- title
- subtitle
- badges
- features
- CTA
- links

No standalone schema required.
