# InstantViral v2

# Document 14.10 — Site-wide QA, Security, Performance & Launch Checklist

Version: 1.0
Status: Final Pre-Launch Specification

## Purpose

Provide one production checklist covering the entire website before launch.

This document validates:

- Functionality
- UX
- SEO
- Accessibility
- Security
- Performance
- Analytics
- Commerce
- Legal
- Deployment

---

## Functional QA

Verify:

- Homepage loads
- All 12 approved service pages work
- Contact page works
- FAQ works
- Track Order works
- Checkout flow works
- Package selection works
- Pricing matches production data
- Internal links valid
- 404 page works

---

## SEO QA

Verify:

- Unique titles
- Unique descriptions
- Canonicals
- Robots
- Sitemap
- Breadcrumb schema
- Organization schema
- FAQ schema
- Review schema eligibility
- No duplicate metadata

---

## Accessibility QA

Verify:

- One H1
- Keyboard navigation
- Focus states
- Contrast
- Alt text
- Form labels
- Screen-reader support
- Responsive layouts

---

## Performance QA

Targets:

- Core Web Vitals monitored
- Optimized images
- Lazy loading
- Code splitting
- No unnecessary JavaScript
- Fast route transitions

---

## Security QA

Verify:

- Environment variables protected
- Admin routes protected
- Input validation
- XSS protection
- CSRF protection
- Rate limiting
- No sensitive data exposed
- Secure headers

---

## Commerce QA

Verify:

- Package prices match InstantViral production data
- Coupons work
- Checkout totals correct
- Successful order creation
- Failed payment handling
- Order tracking privacy

---

## Analytics QA

Verify:

- Consent respected
- No PII sent
- Purchase event verified
- Duplicate prevention
- Debug disabled in production

---

## Legal QA

Verify:

- Privacy Policy
- Terms
- Refund Policy
- Cookie Policy
- Disclaimer
- Contact information
- Refund wording
- No unsupported guarantees

---

## Deployment QA

Before launch:

- Production environment configured
- Domain configured
- HTTPS enabled
- Sitemap submitted
- robots.txt verified
- Error monitoring enabled
- Backups configured

---

## Acceptance Checklist

- Typecheck passes
- Lint passes
- Tests pass
- Production build passes
- No critical issues
- Ready for production launch
