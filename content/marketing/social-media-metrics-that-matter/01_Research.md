# Research — Social Media Metrics That Matter

## Core question
Which social metrics deserve attention?

## Short answer (planning)
Prioritize metrics tied to your goal: reach diagnostics, engagement quality, and conversion proxies. Vanity counts without context create false confidence — cite sources when quoting benchmarks.

## Required sources (from plan)
  - Official platform Insights help docs
  - Google Analytics / Search Console documentation where relevant
